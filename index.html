Palmyra Project

A highly optimized MC server lib written in modern C++, designed to fully utilize all hardware components for maximum performance. AI calculations for mobs run on the GPU, while the anti-cheat system leverages CUDA for enhanced efficiency. However, AMD and Intel GPUs are not supported. Currently, the project remains closed-source.

JFarm

JFarm is a simple, cross-platform farming game built from scratch using SFML as the core engine library. Inspired by games like Family Farm, it lets you build your own barn, grow and harvest crops, feed animals, and produce dairy products to sell. Future updates will add multiplayer support to play with friends and the ability to create and manage your own town. The project is currently closed source.
